По шагам:

1. Установить необходимые библиотеки при помощи requirements.txt
2. Заменить токен бота на актуальный в файле token.txt
3. Запустить проект при помощи файла base.py

Описание:

/start выдает приветственное сообщение, прогноз поддерживает команды 'today', 'tomorrow',
а также доступ по дате в формате `ДД/ММ`

Могут быть проблемы с доступом к погодному сервису при большом числе запросов, т.к. используется
в режиме freemium
